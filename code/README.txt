

Python 3 program for Context Free Grammar (CFGs) and Parsing.

### Usage

----randsentence.py------------------------

	1. It will generate sentences randomly using CFG rule set given in the Chomsky Normal Form (CNF).

	2. Randomly generated sentences are written in  random-sentence.txt file.

----parse.py-------------------------------

	1. Place sentence.txt 

	This file contains five sentences and except for one of the sentence, others belong to given grammar rule.

	You can change the file  (adding new sentences or deleting existing sentences).

	2. This code(parse.py) checks whether a given sentence belongs to the given grammar rule.

	It generates parse tree.
